Project Description
This project is an interactive e-commerce website developed using HTML, CSS, and JavaScript.

The system allows users to:
- View products
- Search and filter products
- Add items to cart
- Modify quantities
- Remove items
- View total cost
- Checkout with validation
- Discover products for final purchase and post sale support.
